## 🎯 Ringkasan Soal

> *Diberikan string yang merepresentasikan satu sisi untai DNA, tugas kita adalah mengembalikan sisi komplementernya di mana "A" berpasangan dengan "T", dan "C" berpasangan dengan "G".*

**Input:** `String`  
**Output:** `String`

```javascript
// Contoh
dnaStrand("ATTGC") // → "TAACG"
dnaStrand("GTAT")  // → "CATA"
```
