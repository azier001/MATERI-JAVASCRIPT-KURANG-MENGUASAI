## 🎯 Ringkasan Soal

> *Diberikan dua bilangan bulat `a` dan `b` (bisa positif, negatif, atau nol — urutan tidak dijamin), jumlahkan semua bilangan bulat di antara dan termasuk keduanya.*

**Input:** `Number`, `Number`  
**Output:** `Number`

```javascript
// Contoh
getSum(1, 0)  // → 1   (0 + 1 = 1)
getSum(-1, 2) // → 2   (-1 + 0 + 1 + 2 = 2)
getSum(2, 2)  // → 2   (kedua angka sama, kembalikan langsung)
```
