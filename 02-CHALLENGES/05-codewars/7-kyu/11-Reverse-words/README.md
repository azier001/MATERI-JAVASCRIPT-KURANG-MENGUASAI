## 🎯 Ringkasan Soal

> *Membalikkan setiap kata dalam sebuah string tanpa menghilangkan atau mengubah posisi spasi asli.*

**Input:** `String`  
**Output:** `String`

```javascript
// Contoh
reverseWords("This is an example!") // → "sihT si na !elpmaxe"
```
