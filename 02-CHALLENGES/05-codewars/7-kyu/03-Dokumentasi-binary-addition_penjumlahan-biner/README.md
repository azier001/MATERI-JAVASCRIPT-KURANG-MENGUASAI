## 🎯 Ringkasan Soal

> *Tulis fungsi yang menjumlahkan dua angka, lalu mengembalikan hasilnya dalam bentuk string biner (basis 2).*

**Input:** `Number`, `Number`  
**Output:** `String`

```javascript
// Contoh
addBinary(1, 1) // → "10"
```
