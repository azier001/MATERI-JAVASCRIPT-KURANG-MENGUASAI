## 🎯 Ringkasan Soal

> _Cek apakah sebuah string memiliki huruf yang berulang (abaikan huruf besar/kecil)._

**Input:** `String`  
**Output:** `Boolean`

```javascript
// Contoh
isIsogram('Dermatoglyphics'); // → true
isIsogram('aba'); // → false
isIsogram('moOse'); // → false
```
