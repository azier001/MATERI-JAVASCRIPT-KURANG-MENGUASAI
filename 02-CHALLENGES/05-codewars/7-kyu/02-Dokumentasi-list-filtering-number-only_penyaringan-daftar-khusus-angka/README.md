## 🎯 Ringkasan Soal

> *Buat fungsi yang menerima array berisi campuran angka non-negatif dan string, lalu kembalikan array baru yang hanya berisi angka (string dibuang).*

**Input:** `Array` (campuran integer non-negatif dan string)  
**Output:** `Array` (hanya berisi integer)

```javascript
// Contoh
filter_list([1, 2, 'a', 'b'])          // → [1, 2]
filter_list([1, 'a', 'b', 0, 15])      // → [1, 0, 15]
filter_list([1, 2, 'aasf', '1', '123', 123]) // → [1, 2, 123]
```
