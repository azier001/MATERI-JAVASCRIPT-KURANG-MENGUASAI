## 🎯 Ringkasan Soal

> *Program untuk menentukan kategori keanggotaan klub (Senior atau Open) berdasarkan usia dan nilai handicap dari sekumpulan calon member.*

**Input:** `Array of Arrays (2D Array)` (tiap array berisi dua integer: `[usia, handicap]`)  
**Output:** `Array of Strings` (tiap string berisi `"Senior"` atau `"Open"`)

```javascript
// Contoh
openOrSenior([[18, 20], [45, 2], [61, 12], [37, 6], [21, 21], [78, 9]]) 
// → ["Open", "Open", "Senior", "Open", "Open", "Senior"]
```
