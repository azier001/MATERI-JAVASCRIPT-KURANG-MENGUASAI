## 🎯 Ringkasan Soal

> _Mencari dan menjumlahkan dua bilangan terkecil di dalam sebuah array yang berisi minimal 4 bilangan bulat positif._

**Input:** `Array<number>`  
**Output:** `number`

```javascript
// Contoh
sumTwoSmallestNumbers([19, 5, 42, 2, 77]); // → 7
```
