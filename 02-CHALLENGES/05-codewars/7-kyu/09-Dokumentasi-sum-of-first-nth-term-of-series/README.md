## 🎯 Ringkasan Soal

> *Buat fungsi yang mengembalikan jumlah dari n suku pertama deret: 1 + 1/4 + 1/7 + 1/10 + ... , dibulatkan 2 desimal dan dikembalikan sebagai String.*

**Input:** `number` (n — jumlah suku)  
**Output:** `string` (jumlah deret, 2 desimal)

```javascript
// Contoh
SeriesSum(1) // → "1.00"
SeriesSum(2) // → "1.25"
SeriesSum(5) // → "1.57"
```
