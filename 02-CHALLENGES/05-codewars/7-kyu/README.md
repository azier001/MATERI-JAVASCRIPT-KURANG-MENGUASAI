## 📖 Tentang Level Ini

Level **7 kyu** mulai memperkenalkan algoritma yang lebih logis. Soal-soalnya berfokus pada:
- Filter & transformasi array
- String manipulation yang lebih kompleks
- Logika kondisional bertingkat
- Penggunaan metode bawaan JavaScript (`.map`, `.filter`, `.reduce`)
