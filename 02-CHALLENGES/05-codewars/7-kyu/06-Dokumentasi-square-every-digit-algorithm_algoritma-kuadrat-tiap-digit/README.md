## 🎯 Ringkasan Soal

> *Fungsi untuk menerima sebuah angka integer, mengkuadratkan setiap digit di dalamnya, lalu menggabungkan hasil kuadratnya menjadi sebuah angka integer baru.*

**Input:** `Number`  
**Output:** `Number`

```javascript
// Contoh
squareDigits(9119) // → 811181
squareDigits(765) // → 493625
```
