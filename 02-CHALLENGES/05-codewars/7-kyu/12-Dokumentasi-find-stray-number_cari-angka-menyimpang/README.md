## 🎯 Ringkasan Soal

> *Diberikan array dengan panjang ganjil yang berisi integer, di mana semua elemen sama kecuali satu angka. Temukan angka yang berbeda tersebut.*

**Input:** `Array of Integers` (panjang ganjil, >= 3)  
**Output:** `Number` (angka yang unik/berbeda)

```javascript
// Contoh
stray([1, 1, 2])                           // → 2
stray([17, 17, 3, 17, 17, 17, 17])        // → 3
stray([8, 1, 1, 1, 1, 1, 1])              // → 8
stray([0, 0, 0, 7, 0, 0, 0])              // → 7
stray([-21, -21, -21, -21, -6, -21, -21]) // → -6
```
