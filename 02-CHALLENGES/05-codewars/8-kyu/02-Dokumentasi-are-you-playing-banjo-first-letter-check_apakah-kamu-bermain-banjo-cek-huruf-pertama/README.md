## 🎯 Abstraksi Tantangan

Tantangan ini menguji kemampuan dasar manipulasi string dan logika kondisional. Kita ditugaskan untuk mengonstruksi sebuah fungsi predikat yang mengevaluasi apakah karakter pertama dari sebuah argumen `name` adalah "R" kapital atau "r" kecil. Fungsi harus mereturn sebuah kalimat terstruktur berdasarkan hasil evaluasi tersebut.

**Input:** `string` (Representasi nama pengguna)  
**Output:** `string` (Pernyataan evaluasi)

```javascript
// Demonstrasi Output
areYouPlayingBanjo("Ringo") // → "Ringo plays banjo"
areYouPlayingBanjo("bravo") // → "bravo does not play banjo"
```
