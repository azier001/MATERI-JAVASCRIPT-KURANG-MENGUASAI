## 🎯 Ringkasan Soal

> Menghitung berapa banyak nilai `true` yang ada di dalam sebuah array, dengan mengabaikan nilai selain `true` (terutama *bad values* seperti `null` atau `undefined`).

**Input:** `Array`  
**Output:** `Number`

```javascript
// Contoh
countSheeps([true, false, true, true]) // → 3
```
