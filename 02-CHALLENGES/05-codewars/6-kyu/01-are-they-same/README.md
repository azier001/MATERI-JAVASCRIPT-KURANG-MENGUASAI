## 🎯 Ringkasan Soal

> *Memeriksa apakah dua array memiliki elemen yang "sama", di mana "sama" berarti elemen di array kedua adalah kuadrat dari elemen di array pertama dengan multiplisitas yang sama (tanpa memperhatikan urutan).*

**Input:** `Array<number>`, `Array<number>` (atau `null`)  
**Output:** `boolean`

```javascript
// Contoh
a = [121, 144, 19, 161, 19, 144, 19, 11]
b = [121, 14641, 20736, 361, 25921, 361, 20736, 361]
comp(a, b) // → true
```
